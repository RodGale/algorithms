# Algorithms

Implementation of various algorithms in Java, with cost analysis.

* Union Find - Dynamic connectivity
    - Quick Find
    - Quick Union
    - Quick Union with improvements
* Three Sum Problem
    - Brute force
    - Using sorting and search
* Binary Search

### Usage

To compile: 

```
javac -d bin -classpath lib/algs4.jar src/*.java
```

To execute: 

```
java -cp bin Main
```
